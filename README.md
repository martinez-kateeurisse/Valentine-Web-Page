# Valentine-Web-Page

A single-page interactive Valentine’s Day website built using HTML, CSS, and JavaScript.
This project features smooth transitions, playful interactions, a memory game, and an animated reward screen — all in one self-contained HTML file.

✨ Features
Passcode-protected entry
Romantic greeting screen
“Will You Be My Valentine?” interaction
Date plan selection (Going Out / Staying In)
3-pair memory matching game
Animated bouquet reward screen
Background music toggle
Floating hearts and confetti effects
Fully responsive design

📁 File Structure
index.html
Everything (HTML, CSS, JavaScript) is contained in a single file.
No external libraries or frameworks required.

🚀 How to Use
Save the file as index.html
Open it in any modern browser
Share or deploy (GitHub Pages, Netlify, etc.)

⚙ Customization
Change the Name

Edit:
- Change the Greeting Messages
- Modify the paragraph inside the greeting section.
Change the Passcode
In the script:
- if (input.value === '111125')
Change Music
- Replace the <audio> source link.
Edit Memory Game Emojis
- Update the gameCards array in JavaScript.

🛠 Built With
- HTML5
- CSS3 (animations, glassmorphism, grid, flexbox)
- Vanilla JavaScript
